# GoGet Business Tracker PWA V2

Versi mobile-first dengan logo dispatch, favicon, PWA icon, job meter dan shift summary meter.

## Apa yang ditambah
- UI phone-friendly, mobile-first dan elak overflow
- App icon/favicons daripada logo dispatch image no. 2
- Job Meter untuk setiap job
- Preview Job Meter sebelum simpan job
- Meter Harian / Summary Meter bila End Shift
- Dashboard target RM3k/RM4k
- Export CSV termasuk score job
- Offline-first PWA

## Formula ringkas meter
Job Meter kira berdasarkan:
- RM per KM
- RM per hour
- total fare/tip

Shift Meter kira berdasarkan:
- net profit vs target harian
- RM per hour
- average job quality
- cost per KM

## Deploy GitHub Pages
1. Upload semua file ke repo GitHub.
2. Settings > Pages.
3. Source: Deploy from branch.
4. Branch: main, folder: root.
5. Save.
